# Deploy Kasm on AWS

## Resources needed

- OS (x86_64/amd64)
  - Ubuntu 22.04
  - Debian 11
  - Oracle Linux 7-9

- AWS instance (hardware requirements)
  - 2 vCPUs
  - 4GB RAM
  - 50GB Storage
- Terraform
- Ansible
- Bash
- NixOS

\* These are subject to change
